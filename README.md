🐉 Kali Linux / Debian / Ubuntu
Download the latest .deb package from Releases.

```Bash
sudo dpkg -i 0xmew-toolkit_3.4_all.deb
```

🎩 Fedora / CentOS / RHEL
Download the latest .rpm package from Releases.

```Bash
sudo rpm -ivh 0xmew-toolkit-3.4-1.noarch.rpm
```
